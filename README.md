# Flight Ticket Price Prediction

This project predicts flight ticket prices using machine learning.  
It includes data preprocessing, exploratory analysis, model training, and evaluation.

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Preprocess the dataset
python data_preprocessing.py

### 3. Train the model
python model_training.py

### 4. Evaluate model performance
python model_evaluation.py

## Project Files

- flight_price_analysis.ipynb — Jupyter notebook for EDA and visualization  
- data_preprocessing.py — cleans and encodes the dataset  
- model_training.py — trains the ML model  
- model_evaluation.py — evaluates performance using MAE, RMSE, R²  
- requirements.txt — dependencies

## Tech Stack
Python, Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
